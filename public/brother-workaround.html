Brother QL-820NWB DK-2205 Setup

⚠️ Known Issue

Brother QL-820NWB printers often show "Wrong Roll Type" when printing from web browsers because the driver doesn't recognize custom page sizes sent via CSS.

✅ Solution: Use Brother P-touch Editor Lite

  1. One-time setup:
    • Open Windows Settings → Printers & Scanners
    • Click "Brother QL-820NWB" → Manage → Printer Properties
    • Go to "Advanced" tab
    • Change "Paper Size" to "USER DEFINED"
    • Set width to 62mm (2.44 in)
    • Set height to 100mm (3.94 in)
    • Click "Apply" and "OK"
  2. Alternative: Use Brother's Software
    • Install Brother P-touch Editor (free from Brother's website)
    • It will add proper paper sizes to Windows
    • The "62mm x 100mm" option will then appear in all print dialogs
  3. Workaround for now:
    • When print dialog opens, choose "Microsoft Print to PDF"
    • Save the hall pass as PDF
    • Open the PDF and print from there (PDFs handle custom sizes better)

Test Print

After setup, test with this button:

TEST LABEL

Width: 62mm (2.4")

Height: 100mm (3.9")

If this prints correctly, your Brother printer is configured properly.
